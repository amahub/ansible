# Ansible-test

Ansibleテスト用プロジェクト  

#### 使用用途
wordpressサーバのプロビジョニング

#### プロビジョニングする環境はを想定
1. Ansibleサーバから開発サーバをプロビジョニング
2. Ansibleサーバから本番サーバをプロビジョニング

#### 以下の内容をプロビジョニングする
1. Nginx
2. WordPress
3. MariaDB
4. Php-fpm
5. ServerSpec

以下リポジトリを、多大に参考とさせて頂いています
<https://github.com/ansible/ansible-examples>
